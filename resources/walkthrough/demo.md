# Try the Fallback Demo

Snipara gives agents durable project context: source-backed docs, decisions, team standards, and reusable memory that survives sessions and model switches.

The main path is **Activate Workspace**, which builds a First Work Brief from your own repository. Use the demo when you are not ready to sign in or the current workspace has no docs.

1. Click **Try Demo Query** below
2. See retrieved source sections, follow-up questions, and context metadata
3. No sign-in, no API key needed

Your agent starts with the project knowledge it needs instead of rebuilding context from scratch.
