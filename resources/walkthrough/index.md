# Index Project Context

Point Snipara at your markdown files and it will index them for source-backed retrieval, shared context, and agent memory workflows.

**Supported formats:** `.md`, `.mdx`, `.txt`

**Two ways to index:**

1. **Auto-detect** — Snipara scans your workspace and offers to index found docs
2. **Manual sync** — Click **Sync Documents** below or right-click any markdown file in the Explorer

Once indexed, your project context is chunked, ranked, and ready for agent queries. Updates are incremental — only changed files are re-processed.
