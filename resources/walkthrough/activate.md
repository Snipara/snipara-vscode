# Build Your First Work Brief

Snipara can start with the project knowledge already sitting in this workspace.

Click **Activate Workspace** to scan bounded starter docs like `README`, `docs/**`, `AGENTS.md`, `CLAUDE.md`, changelogs, and package metadata.

After sign-in, Snipara indexes those files and opens a source-backed **First Work Brief** with the files and questions an AI coding agent should inspect first.

The demo is still available, but the primary proof is your project.
